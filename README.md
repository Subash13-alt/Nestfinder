# 🏠 NestFinder - Property Discovery Platform

NestFinder is a modern property discovery platform that helps users find their perfect home. Built with Next.js 14, MongoDB, and Tailwind CSS.

## ✨ Features

- 🔐 **Authentication** - User login/register with JWT
- 🏠 **Property Listings** - Browse, search, and filter properties
- ❤️ **Favourites** - Save properties you love
- 📅 **Bookings** - Schedule property visits
- 🤖 **Smart Chatbot** - FAQ assistant for property queries
- 👑 **Admin Dashboard** - Manage listings and users
- 📱 **Responsive Design** - Works on all devices
